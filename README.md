# Installation

1. Clone this project

2. Fill in env variables
```sh
cp .env.template .env
```

3. Install dependencies
```sh
npm install -f
```

4. Run the project and navigate to http://localhost:5173
```sh
npm run dev
```

### Run tests

```sh
npm run test
```
